# How to toast bread
---
#### INDGRIDIENTS
* Slice of Bread **(or 2)**
* Butter **(Optional)**
* Jam **(Optional)**
#### YOU WILL NEED
* Toaster
**Recommended:** [Mainstays 2-Slice Toaster](https://www.walmart.com/ip/Mainstays-2-Slice-Toaster-Black-with-6-Shade-Settings-and-Removable-Crumb-Tray-New/1233603801?wl13=2550&selectedSellerId=0&wmlspartner=wlpa&utm_source=chatgpt.com)

---

#### DIRECTIONS
1. Check the toaster. Make sure the toaster is plugged, before starting.

2. Insert the bread

3. Push down the lever to start the toaster.
   + Some toasters may have a setting that chooses the level of strength

4. Wait safely for the toast to pop up. Prompting that it is done.
---
# CAUTION!
 Toast may be hot!

---

5. Place the toast on a plate. Add toppings **(optional)**
6. Enjoy toast.
